ScriptLens for desktop YouTube

Clear transcript review for desktop YouTube.

ScriptLens adds a compact Analyze video button to desktop youtube.com/watch pages. It checks the strongest transcript path, gives you a readable score with context, and opens the workspace only when you want more detail.

  • Transcript quality and provenance are shown with each result.
  • Results stay on the watch page. No separate dashboard needed.
  • Optional recovery only steps in when the local transcript path is not strong enough.

How it works

Transcript-first analysis, inline results.

Scores transcripts on YouTube watch pages. Shows results inline. Opens the workspace when you need more detail.

Transcript first

Start from the spoken content.

ScriptLens prioritizes transcript-class sources before anything else. Title and description are not treated as the default win path.

Local by default

Keep scoring on-device.

The core score runs inside the extension. When stronger recovery is needed, the product can use a configured backend without turning the normal flow into cloud scoring.

Clear handoff

Inline first, workspace second.

Users get the quick answer on the watch page, then move into the side-panel workspace when they need the deeper explanation, transcript evidence, or support path.

Extension surfaces

Three surfaces. One tool.

ScriptLens shows results inline on the watch page. The popup and side-panel workspace give you more detail when you need it.

Transparency

Quality is visible, not hidden.

ScriptLens shows transcript quality and recovery provenance so you can judge whether the score is based on a strong source.

Privacy

Minimal transfer when recovery is used.

In recovery-enabled builds, the extension only sends the YouTube video ID and requested language to the configured backend.
