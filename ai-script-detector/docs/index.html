Chrome Web Store release

One-click transcript review for desktop YouTube.

ScriptLens adds an inline Analyze video button to desktop youtube.com/watch pages. It checks the strongest available transcript path, scores AI-like writing patterns, and shows the verdict before you ever need the full workspace.

  • Transcript-first by default
  • Verdict, score, explanation, and transcript quality in one view
  • Local scoring with an optional localhost helper for harder transcript recovery
See the product Privacy policy
Product surfaces

Simple main flow, deeper tools when needed.

The default path is the inline check. The popup and side panel are there for settings, transcript choices, and a longer report view.

Transcript first

Starts with the spoken content.

ScriptLens tries YouTube transcript paths first. Title and description fallback only happens when the user explicitly allows it.

Local by default

No cloud scoring in the normal flow.

Scoring runs inside the extension. The optional helper is local-only and receives just the YouTube video ID and selected language.

Clearly scoped

Built for desktop watch pages, not everything.

This release is intentionally limited to desktop youtube.com/watch pages. Shorts, m.youtube.com, and generic page scanning are out of scope.

Privacy posture

Transcript text stays on-device in the normal path.

ScriptLens scores locally by default. If the optional helper is enabled, the extension shares only a YouTube video ID and selected language with a process running on the same machine.
