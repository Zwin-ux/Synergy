Effective March 8, 2026

ScriptLens processes YouTube analysis locally by default.

ScriptLens is a YouTube-only Chrome extension. Its job is to analyze the writing style of YouTube transcripts, titles, and descriptions for AI-like patterns. The extension is designed so that scoring happens inside the extension runtime on your device.

What ScriptLens accesses

  • Desktop YouTube watch pages on https://www.youtube.com/*
  • Transcript metadata, transcript text, video title, and video description when those sources are available on the page
  • Local extension settings and compact recent-report summaries stored with Chrome extension storage

What ScriptLens stores locally

  • Your extension settings, such as sensitivity and whether optional transcript recovery is enabled
  • A short recent-report history so the popup and workspace can show past results
  • Optional debug summaries when debug mode is enabled

Recent reports store compact metadata and summaries. ScriptLens does not intentionally keep full transcript text in recent-report storage.

What ScriptLens does not do by default

  • It does not send transcript text to a remote ScriptLens server for scoring
  • It does not collect account identifiers, browsing history outside supported YouTube pages, or advertising identifiers
  • It does not require an account or cloud login

Optional local helper behavior

ScriptLens can optionally use a local helper that you run on your own machine to recover transcripts for videos where the page does not expose a usable transcript path. When this helper path is enabled and used, ScriptLens sends only:

  • the YouTube video ID
  • the requested language, if one is selected

This helper is optional. The extension works without it. The helper is not bundled into the Chrome Web Store package.

Permissions

  • storage to save settings and recent report summaries
  • sidePanel to open the advanced workspace view
  • YouTube host access on https://www.youtube.com/* so the extension can read supported watch-page content

Support and questions

The public support channel for ScriptLens is GitHub Issues. If extension behavior changes in a future version, this privacy page will be updated to match the shipped behavior.
